# futureGEO Resources

This page features every link related to the futureGEO geostationary amateur radio satellite project known to Amateur Radio Daily. The purpose of this page is to elevate the futureGEO project and provide curious hams with the information they need to learn more or get involved with the project. If you're aware of other resources that should be linked here, please drop a note to k4hck@hamweekly.com.

- [Geostationary Microwave Amateur Payload Proposal](https://www.youtube.com/watch?v=_FTvlEyDa1Y) Presented by Frank Zeppenfeldt (PD0AP), ESA at the AMSAT-UK colloquium in 2023
